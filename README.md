# WordWizard
WordWizard to narzędzie napisane w języku Python służące do analizy lingwistycznej i statystycznej tekstów literackich. Program został zaprojektowany z myślą o przetwarzaniu dzieł J.R.R. Tolkiena, oferując precyzyjne mechanizmy ekstrakcji słów, liczb (w tym notacji naukowych) oraz specyficznych sekwencji znaków.
